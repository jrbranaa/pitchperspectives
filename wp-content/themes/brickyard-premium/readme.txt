== Description ==

BrickYard is an easy customizable WordPress blog and magazine theme. It is a fully responsive theme that allows for easy viewing on any device.

== Installation ==

You can install this theme through the WordPress themes installer under "Appearance > Themes - Add New" by uploading the ZIP folder that you got after purchasing the BrickYard Premium theme.
Alternatively, you can download the file, unzip it and move the extracted content via FTP to the "wp-content/themes" folder of your WordPress installation. Then, you will be able to activate the theme.

== Documentation ==

The complete documentation of BrickYard names "documentation.html" and is located in "docs" folder in this theme main folder. It is also accessable via Theme Options Panel after installing and activating this theme.
